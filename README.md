# Nexo Werks Marketing Landing Page

This project is a marketing-focused single-page site for **Nexo Werks**, inspired by the conversion-driven layout style of heyflow.com/ai.

## Run locally

```bash
npm start
```

## Scripts

- `npm start` – serve the static site
- `npm run build` – copy root assets into `docs/` for GitHub Pages
- `npm test` – JavaScript syntax check
